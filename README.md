# ネットミームかるたメーカー

ネットミームをかるた札にできる、ブラウザだけで動くジェネレーターです。

**すべての処理はお使いの端末内で完結します。画像がサーバーに送信されることはありません。**

## 機能

- 頭文字・言葉・画像を入力して、かるた札をリアルタイムプレビュー
- 画像のドラッグ移動・ホイール（ピンチ）ズームでレイアウト調整
- 言葉にスペースを入れると段を区切れる（複数列対応）
- 解説文カードの生成（テキスト入力で自動ON）
- PNG 形式で高解像度（3x）保存
- X（Twitter）の投稿画面を開くシェアボタン
- モバイル: ネイティブシェアシートで画像をLINE等に送信

## 使い方

1. **頭文字** に1文字入力
2. **かるたの言葉** を入力（スペースで段区切り可）
3. **画像** のエリアをタップ・クリックして選択し、ドラッグ・ズームで位置調整
4. 必要に応じて **解説文** を入力（解説カードが追加されます）
5. **プレビュー** を確認して「保存」または「X でポスト」
   - モバイルは「シェア」でネイティブシェアシートから画像を共有できます

## 開発

```bash
npm install
npm run dev
```

```bash
npm run build
```
